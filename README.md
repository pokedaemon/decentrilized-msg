# Дипломная работа
## "Разработка и реализация децентрализованного мессенджера с поддержкой сквозного шифрования и хранения метаданных на блокчейне, используя IPFS и блокчейн"

---

## Безопасность:

1. **Man-in-the-Middle (MITM) атака**
- **Угроза:** Злоумышленник подменяет публичные ключи
- **Защита:** Ключи хранятся в блокчейне (неизменяемые), можно проверить fingerprint
2. **Replay атака**
- **Угроза:** Отправка старых сообщений повторно
- **Защита:** Nonce (одноразовое число) в каждом сообщении + timestamp
3. **Компрометация ключа**
- **Угроза:** Если ключ украли, все сообщения можно расшифровать
- **Защита:** Forward Secrecy (Double Ratchet) - каждое сообщение новый ключ
4. **Metadata leakage**
- **Угроза:** Кто, когда, с кем общается видно в блокчейне
- **Защита:** Частичная - можно использовать временные идентификаторы
5. **Sybil атака**
- **Угроза:** Создание множества фейковых аккаунтов
- **Защита:** Регистрация в Solana стоит денег (лямпорты)

## Блокчейн

### **Почему Solana для этого проекта:**
|Характеристика   | Solana      | Ethereum  |
|-----------------|-------------|-----------|
| Скорость        | 65,000 TPS  | 15-30 TPS |
| Комиссия        | $0.00025    | $1-50     |
| Язык контрактов | Rust        | Solidity  |
| Время блока     | 400ms       | 12 секунд |

#### **Зачем нам блокчейн?**
В блокчейне мы храним **публичную информацию**, которая нужна всем:
- Список пользователей (username → публичный ключ)
- Метаданные сообщений (когда отправлено, TTL, CID из IPFS)
- Proof-of-delivery (подтверждение получения)

---

**Блокчейн даёт:**
- **Неизменяемость** - никто не может подделать историю
- **Децентрализацию** - нет единого сервера
- **Доступность** - любой узел может проверить данные

---

#### **Что храним в блокчейне:**
- Публичные ключи пользователей (для E2EE)
- Metadata: username → public key mapping
- CID (Content Identifier) из IPFS для сообщений
- Timestamps для самоуничтожения
#### **Что НЕ храним в блокчейне:**
- Сами сообщения (они в IPFS, зашифрованные)
- Приватные ключи (только на клиенте)
